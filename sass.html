Wat is Sass?

Sass (Syntactically Awesome StyleSheets) is een scripttaal die CSS uitbreidt. Dit is ontwikkeld in 2006 en geeft developers de mogelijkheid om code te schrijven in de ene taal en deze vervolgens te compileren naar CSS. Sass stelt je in staat om functies te gebruiken zoals variabelen, geneste regels, inline importeren en nog veel meer om het je makkelijker te maken. Het helpt ook om dingen georganiseerd te houden waardoor je stylesheets sneller kan maken. Een aantal functies zal ik in deze presentatie uitleggen.
