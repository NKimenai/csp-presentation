Wat is GSAP?

Greensock Animation API (GSAP) is een animatie library waarmee je simpele maar leuke animaties kunt maken. Het kan veilig worden gebruikt om animaties te maken zonder problemen met browsers, en het is de enige animatie library (voor zover ik weet) die SVG-animaties goed verwerkt.

GSAP compatibiliteit

  • HTML5
  • SVG
  • jQuery
  • Canvas
  • CSS
  • React
  • EaselJS
  • mobiel

Greensock Showcase

Op de website van Greesock is een tabje met showcases.
De huidige showcase die je ziet is www.esperanto.design
Esperanto is een jaar lange reis van Robin Noguier die in 16 landen is geweest om daar getallenteerde web en graphic designers. De webiste animaties zijn gemaakt in met GSAP
Hier staan alle showcases: https://greensock.com/showcase

Opdracht GSAP:

  • Kopieer de afbeelding en de list items en plaats ze in de playground
  • Animeer elk list item anders
  • Fade de afbeelding in
  • En je ben klaar. Goed gedaan hoor
